Create a complete production-ready Python project for Replit using FastAPI that provides AI image and video upscaling.

Requirements:

Project Stack:

- FastAPI
- Uvicorn
- FFmpeg

- aiofiles

Features:

1. Image Upscaling
- POST /upscale-image
- Accept JPG, PNG, WEBP
- Upscale using Real-ESRGAN (4x)
- Return PNG image
- Preserve transparency if available.

2. Video Upscaling
- POST /upscale-video
- Accept MP4, MOV, MKV, AVI
- Automatically:
    - Save uploaded video
    - Extract all frames using FFmpeg
    - Upscale every frame using Real-ESRGAN
    - Merge frames back into video using FFmpeg
    - Copy original audio into output
    - Return final upscaled MP4

3. Performance
- Process frames sequentially to reduce RAM.
- Delete temporary files automatically.
- Show processing progress in logs.
- Support videos up to 5 minutes.
- Handle errors gracefully.
